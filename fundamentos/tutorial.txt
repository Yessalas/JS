Java Scrpit:

A linguagem JS é case sentive (faz diferença entre letras maiusculas e minusculas)

-No JS o ";" é opcional
-Você pode usar "" oi '' não é permitido misturar "' '"
Dica: usar "texto" e 'importações'

Variaveis na linguagem JS são dinâmicamente tipadas.
-string
- number
- boolean

-funvtion
-object

Para criar uma constante usamos: const
Para criar uma variavel podemos usar var ou letras
VAR(obsoleto e inseguro (redeclarar a variavel))
usar sempre o let

concatenação (união de textos com variaveis)
+ (operador de soma) !evitar (tipagem dinâmica)
Usar crase (forma moderada e segura):
'texto ${variavel}'
